# cctv-mk-lab

一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为页面文档的集中存放与版本管理用途。

## 项目简介

cctv-mk-lab 是一个面向通用 HTML 页面归档的 GitHub 仓库。  
它适用于存放、整理和发布独立开发或收集的页面资源，便于后续引用、展示或备份。  
每个 HTML 页面均可独立运行，不依赖外部服务或特定域名环境。

## 目录说明

```
/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面所需的资源文件（可选）
├── docs/           # 相关文档或说明（可选）
└── README.md       # 本文件
```

- `pages/`：每个 `.html` 文件为一个独立页面。
- `assets/`：可包含 CSS、JS、图片等公共资源。
- `docs/`：存放与页面相关的说明文档或使用指引。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间互相独立，无跨页面依赖。
- 归档时建议在文件名或页面内注释中注明版本或日期信息。
- 本仓库不提供自动部署或在线预览功能，页面可通过 GitHub Pages 手动启用。

## 维护说明

- 欢迎提交新页面或改进已有页面，请遵循以下原则：
  - 每个页面保持独立，不引入外部不可控资源。
  - 页面结构清晰，代码可读。
  - 如有资源文件，请放入对应目录并正确引用路径。
- 本仓库为归档性质，不保证频繁更新。
- 如发现页面异常或链接失效，欢迎提交 Issue 或 Pull Request。
